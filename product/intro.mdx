
### MDX（.mdx）
MDX **完全包含**上述 Markdown 语法，并在此基础上增加了组件能力：

```mdx
# 标题

这是一段普通文本，可以**加粗**或*斜体*。

## 使用导入的组件

import Button from '../components/Button'

可以直接使用 React 组件：
<Button variant="primary">点击我</Button>

## 在 Markdown 中混用 JSX

<div style={{ padding: '20px', background: '#f0f0f0' }}>
  <h3>这是一个 JSX 容器</h3>
  <p>里面可以写普通文本，也可以<Button size="small">嵌套组件</Button></p>
</div>

## 导出变量（MDX 特有）

export const meta = {
  title: '文章标题',
  date: '2024-01-01'
}

还可以在列表中使用组件：
- 普通列表项
- <Button>列表中的按钮</Button>
